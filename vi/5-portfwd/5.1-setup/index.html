Cấu hình và Kết nối Ứng dụng Frontend với AWS Backend

Trong phần này, bạn sẽ lấy mã nguồn của ứng dụng frontend, cấu hình để kết nối đến backend AWS vừa tạo, và chạy thử các chức năng chính để xác minh toàn bộ hệ thống hoạt động chính xác.

1. Lấy mã nguồn từ GitHub

  • Mở terminal hoặc command prompt.
  • Chạy lệnh sau để clone mã nguồn: 
    git clone https://github.com/7475-TrungHau/Realtime_QA_Poll.git
  • Di chuyển vào thư mục dự án: 
    cd MYPJ
  • Cài đặt các dependencies: 
    npm install

2. Cấu hình Kết nối Backend

Để ứng dụng frontend có thể giao tiếp với backend, bạn cần cung cấp các thông tin định danh và endpoint của các dịch vụ AWS đã tạo.

Bước 1: Mở file cấu hình

  • Dùng trình soạn thảo code, mở file:
    src/lib/amplify-client.ts

Bước 2: Cập nhật thông tin cấu hình

Bạn sẽ thấy một đoạn code với các giá trị placeholder. Hãy thay thế chúng bằng thông tin từ tài khoản AWS của bạn.

// src/lib/amplify-client.ts
import { Amplify } from 'aws-amplify';
// ...
Amplify.configure({
  Auth: {
    Cognito: {
      userPoolId: 'YOUR_USER_POOL_ID', // <-- Thay bằng User Pool ID của bạn
      userPoolClientId: 'YOUR_USER_POOL_CLIENT_ID', // <-- Thay bằng App Client ID
    }
  },
  API: {
    GraphQL: {
      endpoint: "YOUR_APPSYNC_GRAPHQL_ENDPOINT", // <-- Thay bằng URL Endpoint của AppSync
      region: 'YOUR_REGION', // <-- Thay bằng Region của bạn, ví dụ: ap-southeast-1
      defaultAuthMode: 'apiKey',
      apiKey: "YOUR_APPSYNC_API_KEY", // <-- Thay bằng API Key của AppSync
    }
  }
}, { ssr: true });

3. Lấy các giá trị cấu hình từ AWS

  • userPoolId:
    Truy cập dịch vụ Amazon Cognito trên AWS Console.
    Chọn User Pool: RealtimeQAPlatform-UserPool.
    User Pool ID nằm ngay trên trang tổng quan.

  • userPoolClientId:
    Trong User Pool, chọn App Clients.
    Chọn App Client của bạn. App Client ID nằm ngay trên trang tổng quan.

  • region:
    Ví dụ: ap-southeast-1

  • endpoint:
    Truy cập dịch vụ AWS AppSync trên AWS Console.
    Chọn API của bạn, vào mục Settings. Tất cả các thông tin này đều có trong phần API Details.

  • apiKey:
    Trong AppSync, chọn Settings → cuộn xuống dưới phần API Keys (Lưu ý chọn API key còn hạn).
